# matchmaker-jp# Matchmaker for the Web

This project is a simple web application that determines compatibility between two people based on their responses to relationship-style questions.

## How it Works

Users answer five statements using a 1–5 scale:

1 = Strongly Disagree  
5 = Strongly Agree  

The application compares the user's answers with predefined "true love" answers. The difference between answers is calculated using the absolute value formula.

Final compatibility score:

100 − (sum of answer differences)

## Features

- 20 question bank
- Randomly selects 5 questions each time
- One question shown at a time
- Input validation
- Recap of answers
- Compatibility percentage result
